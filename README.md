# Autonomous AI Browser Agent 🤖

Автономный агент, управляющий браузером через Playwright и OpenAI API.
Использует паттерн **ReAct** (Reasoning + Acting) для выполнения сложных многошаговых задач.

## 🚀 Особенности
- **Vision:** Анализирует DOM-дерево и скриншоты для принятия решений.
- **Resilience:** Умеет восстанавливаться после ошибок (закрывать попапы, искать альтернативные пути).
- **Stealth:** Использует реальный профиль Chrome для обхода анти-бот систем (Яндекс/Cloudflare).

## 🛠 Технологии
- **Language:** Python 3.11 (выбран для скорости прототипирования AI-фич).
- **Core:** Playwright (Async), OpenAI SDK.
- **Models:** gpt-5-mini.

## ▶️ Запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Добавьте `OPENAI_API_KEY` в `.env`
3. Запуск: `python main.py`

## 👨‍💻 Пример работы
> User: "Зайди на Яндекс Лавку и добавь 2 хотдога"
> Agent: *Сканирует -> Ищет -> Добавляет -> Проверяет корзину -> Готово.*
